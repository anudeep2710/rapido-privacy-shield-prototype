# Rapido Privacy Shield Prototype

Clickable prototype for a Product Management Intern case-study submission.

## Flow

- Booking confirmation with Privacy Shield notice
- Captain assignment with protected call/chat
- Pickup guidance with a time-boxed data lease
- Ride completion with Captain access expiry
- Ride privacy details with misuse report, deletion request, and access summary

## Local Preview

Open `index.html` in a browser.

## Deployment

This is a static site. Vercel can deploy it with no build step.
